# 🐞 Bug Report: Date Picker Misaligned on iOS Safari

**Summary:**  
On mobile Safari, the `<input type="date">` field appears cropped and cannot be selected.

**Root Cause:**  
CSS shadow DOM styling not supported by Safari's rendering engine.
